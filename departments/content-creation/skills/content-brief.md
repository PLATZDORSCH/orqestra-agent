# Content Brief Template

Use this structure when planning a content piece:

## 1. Content Overview
- **Type**: blog post / social media / newsletter / whitepaper
- **Target audience**: who will read this?
- **Goal**: inform / persuade / entertain / educate
- **Tone**: professional / casual / technical / conversational

## 2. Research Sources
- List relevant wiki pages to reference
- External sources and data points needed
- Competitor content to differentiate from

## 3. Structure
- Headline (compelling, clear)
- Introduction (hook + context)
- Body sections (3-5 key points)
- Call to action or conclusion

## 4. SEO / Distribution
- Target keywords (if applicable)
- Distribution channels
- Recommended publish date

## Output
Save the brief and draft to `wiki/ergebnisse/YYYY-MM-DD-slug.md` with metadata:
- title, category (drafts), tags, status (draft), content_type
