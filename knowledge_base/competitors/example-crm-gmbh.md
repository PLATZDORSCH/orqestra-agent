# Example CRM GmbH

## Profile

- **Headquarters**: Munich, Germany
- **Founded**: 2019
- **Employees**: ~120
- **Funding**: Series B (EUR 15M, 2023)
- **Customers**: ~800, predominantly SMEs (50–500 employees)

## Product

Cloud CRM specialized for the German-speaking SME market. Key features:
- GDPR-compliant hosting in Germany
- Native integration with DATEV and SAP Business One
- German-language UI and support

## Strengths
- Strong localization for the DACH region
- DATEV integration as a unique selling point
- Short implementation times (avg. 4 weeks)

## Weaknesses
- No international presence
- Limited API ecosystem compared to Salesforce / HubSpot
- Marketing automation still basic

## Pricing
- Starter: EUR 29/user/month
- Professional: EUR 59/user/month
- Enterprise: on request
